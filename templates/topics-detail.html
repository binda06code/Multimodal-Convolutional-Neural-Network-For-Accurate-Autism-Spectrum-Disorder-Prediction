Introduction to DetectASD

DetectASD is an innovative website designed to assist in early autism diagnosis for children aged 6 to 36 months. Using advanced machine learning and artificial intelligence, DetectASD offers a faster and more accurate approach compared to traditional methods.

>Our method combines advanced image analysis and eye-tracking technology to examine facial expressions and eye movements. The image dataset includes 2,939 images with various facial expressions, while the eye-tracking dataset includes measurements like pupil size, gaze direction, age, and gender, providing a comprehensive assessment.

DetectASD employs highly accurate models:

  • CNN models trained with VGG-19, Xception, and MobileNet architectures. The Xception-SeLU model achieved 87.5% accuracy.
  • KNN model with k=2 trained on eye-tracking data, achieving 99% accuracy.

Users can choose to use the facial analysis, eye-tracking analysis, or both for a detailed multimodal diagnosis. The web application supports image inputs (PNG, JPG, JPEG) and eye-tracking data (CSV) to predict ASD.

Built with HTML, CSS, and JavaScript for the front end, and Python for model integration, DetectASD utilizes datasets from Kaggle and Figshare to ensure robust and dependable results.
